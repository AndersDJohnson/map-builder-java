# map-builder-java
Builder for Maps.

## Use

```java
Map<String, String> map = new HashMapBuilder<>()
  .put( k1, v1 )
  .p( k2, v2 )
  .putAll( otherMap )
  .put( mapEntry )
  // ...
  .build();
```
